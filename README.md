# murnana-claude-code-plugin

murnana が個人利用を目的として管理する Claude Code プラグインマーケットプレイスです。

> **注意:** このマーケットプレイスおよび各プラグインは個人用途で作成・公開されています。  
> **動作保証は一切致しかねます。** 自己責任のもとでご利用ください。

## マーケットプレイスの追加

```
/plugin marketplace add murnana/murnana-claude-code-plugin
```

## プラグインのインストール

```
/plugin install <plugin-name>@murnana-claude-code-plugin
```

現在公開中のプラグインはありません。
